Pertama-tama <br/>
Buat folder baru dengan nama backend didalam rootfolder/nama project <br/>
Didalam folder backend tambahkan file dengan nama server.js
Lalu
``npm init`` <br/>
<br/>
package.json
``
"scripts": {
    "start": "node backend/server.js",
    "server": "nodemon backend/server.js"
  },
``
